# LangGraph代码结构

```python
workflow = StateGraph(StoryState)

workflow.add_node(
    "script",
    script_agent
)

workflow.add_node(
    "character",
    character_agent
)

workflow.add_node(
    "storyboard",
    storyboard_agent
)

workflow.add_node(
    "image",
    image_agent
)

workflow.add_node(
    "voice",
    voice_agent
)

workflow.add_node(
    "video",
    video_agent
)

workflow.add_edge(
    START,
    "script"
)

workflow.add_edge(
    "script",
    "character"
)

workflow.add_edge(
    "character",
    "storyboard"
)

workflow.add_edge(
    "storyboard",
    "image"
)

workflow.add_edge(
    "image",
    "voice"
)

workflow.add_edge(
    "voice",
    "video"
)

workflow.add_edge(
    "video",
    END
)
```

# PostgreSQL设计

## story

```sql
CREATE TABLE story
(
    id UUID PRIMARY KEY,

    title VARCHAR(255),

    genre VARCHAR(50),

    prompt TEXT,

    status VARCHAR(30),

    created_at TIMESTAMP
);
```

---

## character

```sql
CREATE TABLE character
(
    id UUID PRIMARY KEY,

    story_id UUID,

    name VARCHAR(100),

    appearance JSONB,

    personality JSONB,

    avatar_url TEXT
);
```

---

## episode

```sql
CREATE TABLE episode
(
    id UUID PRIMARY KEY,

    story_id UUID,

    episode_no INT,

    script TEXT
);
```

---

## storyboard

```sql
CREATE TABLE storyboard
(
    id UUID PRIMARY KEY,

    story_id UUID,

    scene_no INT,

    prompt TEXT,

    camera VARCHAR(50),

    duration INT
);
```

---

## task

```sql
CREATE TABLE task
(
    id UUID PRIMARY KEY,

    story_id UUID,

    status VARCHAR(50),

    progress INT,

    current_step VARCHAR(50),

    error_message TEXT
);
```

# Script Agent

职责：

```text
创意
↓
剧情大纲
↓
章节规划
↓
完整剧本
```

Prompt：

```text
你是一名专业漫剧编剧。

根据用户需求生成：

1.人物设定

2.剧情大纲

3.章节规划

4.对白
```

```json
{
  "characters":[],
  "outline":"...",
  "episodes":[]
```
