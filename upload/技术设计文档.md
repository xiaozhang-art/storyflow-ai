# 3. 数据库设计

## story

```sql
CREATE TABLE story (
    id UUID PRIMARY KEY,

    title VARCHAR(255),

    prompt TEXT,

    genre VARCHAR(50),

    total_episode INT,

    status VARCHAR(50),

    cover_image TEXT,

    created_at TIMESTAMP,

    updated_at TIMESTAMP
);
```

---

## character

```sql
CREATE TABLE character (

    id UUID PRIMARY KEY,

    story_id UUID REFERENCES story(id),

    name VARCHAR(100),

    gender VARCHAR(20),

    age INT,

    appearance JSONB,

    personality JSONB,

    avatar_url TEXT,

    lora_model TEXT,

    created_at TIMESTAMP
);
```

appearance:

```json
{
 "hair":"black",
 "body":"fat",
 "cloth":"white shirt"
}
```

---

## episode

```sql
CREATE TABLE episode (

    id UUID PRIMARY KEY,

    story_id UUID REFERENCES story(id),

    episode_no INT,

    title VARCHAR(255),

    summary TEXT,

    script TEXT,

    created_at TIMESTAMP
);
```

---

## scene

```sql
CREATE TABLE scene (

    id UUID PRIMARY KEY,

    episode_id UUID,

    scene_no INT,

    prompt TEXT,

    camera VARCHAR(50),

    duration INT,

    image_url TEXT,

    audio_url TEXT
);
```

---

## task

```sql
CREATE TABLE task (

    id UUID PRIMARY KEY,

    story_id UUID,

    status VARCHAR(30),

    progress INT,

    current_step VARCHAR(50),

    retry_count INT,

    error_message TEXT,

    created_at TIMESTAMP
);
```

# 5. Agent设计

---

# Requirement Agent

职责：

用户需求分析

输入：

```text
胖子甄大卫逆袭校花莲花
校园
20集
```

输出：

```json
{
 "genre":"校园逆袭",
 "theme":"成长",
 "episodes":20
}
```

---

# Script Agent

Prompt

```text
你是一名专业短剧编剧。

要求：

1 生成人物设定

2 剧情大纲

3 每集概要

4 完整对白

输出JSON
```

输出：

```json
{
 "outline":"",
 "episodes":[]
}
```

---

FFmpeg命令

```bash
ffmpeg \
-loop 1 \
-i image.png \
-i audio.wav \
-shortest \
-c:v libx264 \
output.mp4
```

---

多个镜头合并

```bash
ffmpeg \
-f concat \
-i list.txt \
output.mp4
```

# 9. OpenAPI设计

---

## 创建项目

```http
POST /api/story
```

Request

```json
{
 "title":"逆袭校花",

 "prompt":"胖子逆袭校花"
}
```

Response

```json
{
 "story_id":"xxx"
}
```

## 启动生成

```http
POST /api/story/{id}/generate
```

---

## 查询状态

```http
GET /api/task/{id}
```

返回：

```json
{
 "status":"running",

 "progress":60,

 "step":"image_agent"
```
