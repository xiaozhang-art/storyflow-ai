# 📘 09 - Memory / Vector Store 设计（V1 / Agent 长期记忆系统）

---

# 1. 设计目标

Memory System 要解决 5 个核心问题：

---

## ❌ 没有 Memory System

* 角色前后不一致
* 剧情越写越崩
* session 一断就失忆
* 多 Agent 之间信息不共享
* 只能靠 prompt 临时补救

---

## ✅ 有 Memory System

* 角色一致性稳定
* 世界观长期保持
* 多 session 可恢复
* 多 Agent 可共享认知
* Skill 可“学习”

---

# 3. Memory 类型定义

## 3.1 Working Memory（短期）

```python id="mem_02"

  "session_id": "Sa",
  "content": {...}
```


## 3.3 Conversation Memory（结构记忆）


👉 整个故事级别记忆

---

## 3.4 Long-term Memory（长期）

